# Quickstart: Interface Components — Single-File UI

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Safari, Edge)
2. Navegue pela grade de figurinhas organizada por países
3. Clique em uma figurinha para marcá-la como "possuída"
4. Use os botões `+` e `-` para ajustar a quantidade de repetidas
5. Use as abas "Todas / Faltantes / Possuídas / Repetidas" para filtrar a grade
6. No rodapé, veja o relatório gerado automaticamente
7. Clique em "Copiar" para copiar o texto formatado para o WhatsApp

## Pré-requisitos

- Navegador moderno com suporte a ES6+, CSS Grid, Flexbox e `navigator.clipboard`
- Nenhuma instalação, servidor ou build necessários

## Estrutura do Projeto

```
index.html   ← Arquivo único com HTML + CSS + JavaScript
```
